# MediaSelector
多媒体文件选择，基于https://github.com/LuckSiege/PictureSelector

1、包含图片压缩功能

2、兼容Android P、Q

3、视频录制和裁剪使用阿里云短视频SDK基础版，SDK需要付费，可体验一段时间，可参考：https://www.alibabacloud.com/help/zh/doc-detail/53407.htm?spm=a2c63.p38356.b99.374.3eb85c5ah8NBsN
